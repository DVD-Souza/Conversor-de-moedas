<h1>Conversor de Moedas</h1>
<div>
  <h2>Sobre o projeto:</h2>
  <p>O "Conversor de Moedas" é uma aplicação Java, criada com o intuito de cumprir o challenger de mesmo nome realizado pela plataforma Alura.</p>
  <p>A aplicação consiste em consumir uma API de conversão de moedas para realizar a conversão de um valor especificado pelo usuário entre uma moeda (como o Dolar(USD)) e outra (como o Real Brasileiro(BRL)).</p>
</div>
<div>
  <h2>Tecnologias aplicadas:</h2>
  <uL>
    <li>Java</li>
    <li>Gson (Biblioteca)</li>
    <li>ExchangeRate (API)</li>
  </uL>
